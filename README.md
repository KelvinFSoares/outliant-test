## Tech stack: Vite + Vitest + React + TS + Tailwind + Ant design + Testing Library

## Running locally

Clone the project

```bash
  git clone https://github.com/KelvinFSoares/outliant-test
```

Enter the project directory

```bash
  cd outliant-test
```

Install the dependencies

```bash
  npm install or yarn
```

Start the project

```bash
  npm run dev
```
